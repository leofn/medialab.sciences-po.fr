1. LOGIN

To login use the link "Admin" in the websites footer or follow this url http://medialab-dev.sciences-po.fr/wp-admin

Once you are on the login page, you can use either : The super admin login: adminscpo - medialab2012 Your LDAP login Your wordpress login

And now, welcome on the wordpress dashboard!

2. HIGHLIGHTED POSTS

The higlighted posts are displayed into what we call the "Editorial Zone". There are 3 levels of importance.

2.1 ADD A POST AND CHOOSE ITS PLACE

To add a post in what we call the "Editorial zone", on your dashboard, click on Posts then click on Add new Now you can edit your new post.

Fill the Title , Content and Excerpt fields with your content

Look for the block named Custom fields , select Subtitle in the dropdown box and in the field right next to this box, put your subtitle (This is what will be displayed as a h3 markup into the Editorial Zone) don't click on Add Custom field

Find the block Categories and select the level of importance of your post :

  • Zone Edito // level 1 = Biggest box of the editorial zone

  • Zone Edito // level 2 = Medium sized box of the editorial zone

  • Zone Edito // level 3 = Smallest box of the editorial zone Be sure to have checked only one category

Now find the block named Featured image and click on Add featured image , in the popup box click on Select files and select your image, in the new window in the box, click on Use as featured image then close the window.

Now click on Publish

2.1.1 ADD A VIDEO AS A BACKGROUND INTO THE EDITORIAL ZONE (ZONE 1 ONLY)

To add a video, just use the custom field named post_video and paste the embed code you picked up on vimeo.

Then click on Publish

WELL DONE YOU JUST ADDED A POST INTO THE EDITORIAL ZONE !

2.2 REMOVE A POST FROM A ZONE

To remove a post from a zone, simply display the list of all posts by clicking on Posts in the sidebar menu of WP, select the wanted post and click on Quick edit, change the status by clicking on Draft. The post will not be published anymore and won't appear into the zone but you can still modify it.

3. TOOLS/PEOPLE/PROJECTS/R&D POSTS

This type of posts are custom post types, you create an post which is an object and you link it with a taxonomy. For example: Create a Project object and link it with a Person and/or a Tool.

3.1.ADD A POST AND LINK IT TO A TAXONOMY

Choose the type of post you want to create in the wordpress menu on your left.

For example: To create a Project Post, click on Project then, click on Add projet Now you can edit your new project.

3.1.1 EDIT YOUR CUSTOM POST

Choose the type of post you want to create in the wordpress menu on your left.

To create a Project Post, click on Project then, click on Add projet Now you can edit your new project.

Fill the Title , Content and Excerpt fields with your content

IF YOU ARE CREATING A TOOL POST find the Custom Field block and select tool id and enter the title of the tool in lowercase without spaces

Now find the block named Featured image and click on Add featured image , in the popup box click on Select files and select your image, in the new window in the box, click on Use as featured image then close the window.

To link your project with a person, find the block People on your left and click on the name of the people you want to link to this project, now fill the right block "Projets" with the name of the projects.

To link a project with a publication, when your are in the page of the project, find the block Publications on your right and fill it with the name of the publication you want to link to this project.

Creating a People post ? Take a look on what custom fields you need to fill.

3.1.2 CREATE A TAXONOMY TO LINK POST OBJECTS

In the wordpress menu, once you clicked on a custom post (for example: Project), just below there is other vocabularies (People, RandD), these vocabularies are ment to be linked to a project post, it is what you see in the sidebar on the website.

You already know how to link some of these vocables to a post, but if you want to create more vocables into (People or RandD) here is how to do it:

I want to create a new person to link him/her with my new project post.

Once my post is written (even as a draft), I go to the sub-section of the Project menu and I click on People. A list appears with all the names on the right and textfields on the left.

To create a new Person, I just need to fill the textfields Name, Slug and Description

Then I click on Add Person

Now you can return on your project post and link it with your new person.

And now click on Publish .

CONGRATS YOU JUST CREATED A NEW CUSTOM POST

3.2.SPECIFIC ACTIONS ON CUSTOM POSTS

3.2.1 ALL CUSTOM POSTS

In all custom posts you can fill the custom fields named date debut et date fin, it is used to sort objects into the pages. For example, in the People page, people are sorted by descending order using the value or date fin as a reference. If date fin of a person named X is further in time than a person named Y, Mr. X will appear first.

3.2.2 TOOL POSTS

The tools posts are a little bit different than the other custom posts, it calls datas from an external json file.

To create a tool post, fill the Title field and fill the Tool id custom field with the name of the directory containing the json file. (http://lrrr.medialab.sciences-po.fr/tools/NAME OF THE DIRECTORY), it will look for the meta.json file supposed to be contained in the directory.

3.2.3 PEOPLE POSTS

The date_debut et date_fin fields are no longer required. To specify whether a person is currently working or not, just fill the active textfield. If the person is active then enter 1 in the value field, if he or her is not active anymore then enter 0 in the value field.

3.2.4 PUBLICATION POSTS

UPDATE June 2014 : There is a new taxonomy for the publications, here are the terms that can be associated to a publication : "journal articles", "conferences", "books", "thesis", "working papers" are display in the left column,
"websites", "datascapes", "Maps & Viz" are display in the right column.
If you create a new publication, find the "publications_types" on the right side and fill in it the name of the pubication type : one (or more) of the terms listed above.

UPDATE June 2014 : On the publication page, you can also add the direct url of the publication (for example, a link to a website, or a pdf for a journal articles...) to the preview of the publication post.
This url will appears when you click on the "Publication" menu and when you consult a publication post.
If you create a new publication, find the custom fields in the page, in the drop-down menu choose "remote_url" and fill the "value" field with a valid url beginning by http://.

4. MENUS

In this website, you have two menus (the main navigation on top of the page, and the footer navigation). In order to Edit these menus, you have to click on Appearance > Menus in the wordpress menu.

On the left of the screen is everything to can add to your menus: Static Pages, Links to an external URL, Links to posts or custom posts. On the right of the screen are the menus. Each menu is in a tab on top of the box, to edit one menu in particular, just click on the wanted tab.

Once you edited your menu click on Save Menu, be careful it only saves the current tab, if you edited the other menu you have to click on its tab and click on Save Menu again.

5.CREATE/EDIT A STATIC PAGE

To create or edit a static page, just click on Pages in the wordpress menu.

In the list DON'T TOUCH THE "TOOLS", "PROJECTS", "PEOPLE" and "RandD" pages. You can edit the other pages and create new ones. Just fill the fields Title and Content, you can add pretty much any code (google maps, video...) into the Content section. You can also add a Featured image

6.MISC

6.1.ADD A VIDEO TO A POST/PAGE

To add a video to a post just paste this following code anywhere in the Content section of a page or a post:

[ylwm_vimeo]THE ID OF THE VIDEO[/ylwm_vimeo].

You can also add optional parameters to your video:

[ylwm_vimeo height="400" width="600" class="MyCustomClass"]20689704[/ylwm_vimeo]

6.2 EDIT HTML/CSS/PHP TEMPLATES

To edit the HTML/CSS/PHP pages, click on Appearance then Editor_ in the wordpress menu. Edit the page and click on Save Changes
